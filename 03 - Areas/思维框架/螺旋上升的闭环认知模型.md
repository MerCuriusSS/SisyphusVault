# 1.[[基础概念|核心定义]]：

- 定义：螺旋上升闭环的个人认知模型
- 核心目标：摆脱「认知困境」，提升个人认知能力


# 2.[[基础概念|本质/底层逻辑]]：

以“学习-实践-知识库-复盘”的自循环、目标驱动、隐性知识显式化、模式化为基础，摆脱「认知困境」，实现能力的“复利增长”和高效输出的认知模型。
「定下目标，不断通过“学习-实践-知识库-复盘”的方式，实现知识的运用和举一反三，并要求能口头转述或文字记录。」
# 3.边界：

- 适合有方向、肯动手、勤于反思、对自己的长期发展有严格要求的人


# 4.执行流程：

- 设立目标并建立知识体系：
	- 想做什么？
	- 目前差距在哪里？
	- 建立目标所在领域的知识树。
- 学习
	- 带着目的去掌握相关的理论，存入自己的知识库。通过「归纳」将零散的知识整理成有序的规则；通过「演绎」
- 实践与解决问题
	- 事物认知型：使用公式：「认知=静态结构+动态发展+环境」透视任何复杂的事物。
	- 问题解决型：
		- 定义问题：（现状+目标+限制条件）
- 复盘与显性化
- 沉淀与更新
