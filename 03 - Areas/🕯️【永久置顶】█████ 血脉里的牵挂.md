# 一、老爸弥留阶段

🔴 开端：

- ***现象***：老爸开始说一些怪话「坐标不一样了，从167变到168，张沙发好高」、开始大喘气、发出嘻嘻嘘嘘的声音、手脚冰凉、上半身直冒汗、两眼时不时反白。
- ***感受***：从不以为意，逐渐变得紧张、惶恐、不知所措

🔴 叔父见面：

- ***现象***：大叔握着老爸手，老爸对其一字一字慢慢地、开心且夸耀地说着幽默的话「细路，咁靓仔啊」，大叔瞬间双眼发红；接着就是哥键、三叔、二舅轮流问候老爸，老豆一一笑着回应。
- ***感受***：从看到大叔发红的双眼开始，眼泪再也止不住地从眼睛里蹦出来，瞬间觉得很伤心、很痛苦、很难受，偷偷地躲到房间里无声哭泣。

🔴 选择：

- ***现象***：老爸时不时地强调自己很热；二叔跟老爸提议下一楼，老爸直摇头，一边威严又坚定地要求坐在二楼不准动，听到的姑妈阿妈任何话语都敏感地质问目的；包括我在内的四姐妹每人轮流舀一勺蜜糖水喂给老爸喝，轮到作为幺仔的我时，还要求我舀多一口给他喝，他一边喝，一边开心地对我笑
- ***感受***：非常非常羞愧、悲痛，真真切切地感受到老爸对我这个“儿子”身份的偏爱，但我“天真”的处世之道却做不动让他放心——越不放心，越疼爱。而老爸现在真的要离开了。第二次无声哭泣。

🔴 遗言：

- ***现象***：一字一句地严肃叮嘱哥健一定要多生几胎，因为太公跟他交代过一定要多子多孙，枝繁叶茂，一代传一代；称赞姑妈是一个“好”家姐，同时也叮嘱她把握“好”的分寸（不要好心办坏事？）；我们四姐妹答应老爸会好好照顾阿妈、啊嘛，我会大个仔，会生性的，不会让他再担心的。老爸闭着眼睛，一边喘着气，一边听着笑着，嘴里重复地说着“好！好！”。
- ***感受***：第一次深刻地体会到“家族传承”这个词在老爸心目中的份量，第一次触碰到这个曾经轻如耳旁风实则千斤重的承担。这一刻我意识到交棒的时候到了，不管有没有准备，都要坚定地允下对老爸的承诺——一定要做个“叻仔”，好好尽好孝心，努力照顾好阿妈、啊嘛，又要做一个合格的“家主”，尽到自己的家族责任。

🔴 睡觉：

- ***现象***：老爸突然喊道“好！睡觉！睡觉啦！睡觉啦！”，然后慢慢躺在沙发上，一边喘气一边说着梦话一样：“不要锁门啊，不要走啊，等我啊”。然后就是时而想坐着，时而想躺着，时不时说着“舒`~~`服！”
- ***感受***：对老爸的举动很愕然，看着老爸辛苦的样子又很心酸。

🔴 熄灯：

- ***现象***：老爸的身体慢慢没有力气，头都抬不起来了，头上一直冒汗，呼吸也越来越促，慢慢地他把鼻孔的氧气口甩掉，眼睛紧闭，单纯用口喘着粗气，由急促渐渐转慢，口一开一合的频率也越来越慢。2秒一次、3秒一次、5秒一次，直到停在了张开的那一刻。瞬间，耳边传来一声声“西方接引，阿弥陀佛~”，宣告着老爸的生命走到了尽头，他~摆脱了痛苦，也永远地离开了我们。
- ***感受***：眼睛盯着老爸的嘴一张一合的时候，我觉得整个世界都不像是真的，我的意识逐渐从身体抽离，视角从“嘴巴”抬高到整个脸，然后注视着这张苍白、瘦成皮包骨的脸，我不敢相信眼前的这个人就是自己的父亲，熟悉却又那么陌生，他就这样永远地离开我了，没有任何预备。我也没有感到什么窒息般的悲痛情绪，只觉得很怅然，内心空荡荡似的。


# 二、无声的传承

🔴 ***事***：
	- 看事情要全面，要留意周围环境，不要局限在某一部分。
	- 做事情要谨慎，既要提前做好万全准备，也要考虑后果与风险。

🔴 ***人***：
	- 做人要胆大心细，不要怯，要勇于承担责任；不要撒撒滚，要淡淡定定。
	- 好好照顾家里人，尽好孝心，维护家庭和睦。
	- 扮演好“家主”的角色，兑现“梁氏源远流长”的承诺。
	- 严于律己，保持良好的生活习惯。

# 三、身影的力量

🔴 ***记住他笑着叹气的表情***——这是他对你不够醒目的规训。

🔴 ***记住他笑着点头的表情***——这是他对你表现“叻仔”的肯定。

🔴 ***记住他笑着说好的表情***——这是他对你允诺照顾好家人的心安。

🔴 ***记住他佝偻身子挥手告别的背影***——这是我对父亲彻底放下、完全释然的最好想象！！！
